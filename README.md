# AI Company — 仮想カンパニーで業務を組織的に回すClaude Codeプラグイン

Claude Code上に仮想組織を構築し、秘書が窓口となって業務を振り分け・管理するプラグインです。
2つの質問に答えるだけで組織が立ち上がり、仕事が進むにつれて部署が自然に増えていきます。

## インストール

Claude Codeを起動した状態で、以下の2ステップを実行してください。

**Step 1: マーケットプレイスを追加**
```
/plugin marketplace add pontarou-re/ai-company
```

**Step 2: プラグインをインストール**
```
/plugin install company@ai-company
```

## クイックスタート

```
/company
```

秘書がオンボーディングを開始します。2つの質問に答えるだけで、仮想組織が立ち上がります。

## コマンド例

| コマンド | 説明 |
|---------|------|
| `/company` | 運営モード開始 |
| 「ダッシュボード」 | 全部署の概要+KPIを表示 |
| 「週次レビュー」 | 今週の成果を集計 |
| 「実績レポートを作成」 | 期間指定で成果レポート生成 |
| 「業種テンプレートを使う」 | 業種別の部門構成を一括セットアップ |
| 「判断を整理して」 | 意思決定フレームワークで比較・分析 |
| 「Notionにエクスポート」 | Notion向け形式で出力 |
| 「Obsidianにエクスポート」 | Obsidian向け形式で出力 |

## 機能一覧

### 秘書室（常設）

秘書が全ての窓口。ユーザーは部署を意識する必要がありません。

- **TODO管理**: 優先度（高/通常/低）・期限付きのタスク管理。日次ファイルで運用
- **壁打ち・相談**: 対話で深掘りし、結論が出たらnotesに保存
- **メモ・クイックキャプチャ**: 思いついたことをinboxに即記録
- **意思決定ログ**: 判断の記録を自動保存。後から振り返り可能

### 部署の自然な追加

同じ領域のタスクが2回以上繰り返されると、秘書が部署の新設を提案します。
明示的に「○○部門を作って」と言えば即座に作成。

利用可能な部署:

| 部署 | 役割 |
|------|------|
| **リサーチ** | WebSearch必須・出典付与・未確認正直・一次資料優先の4原則でファクトベースの調査 |
| **レビュー** | リサーチ成果物の品質検証。A-D評価フォーマットで出典・数値・論理をチェック |
| **マーケティング** | SEO・集客・エンゲージメント戦略。Shorts導線設計、競合分析 |
| **PM** | プロジェクト管理。マイルストーン・進捗・スケジュール管理 |
| **開発** | 実装・設計・アーキテクチャ・デバッグ |
| **経理** | 請求・経費・売上・確定申告・インボイス |
| **営業** | クライアント・提案・見積・案件・商談 |
| **クリエイティブ** | デザイン・ロゴ・バナー・ブランド |
| **法務** | 契約書の精査・リスク検出・修正提案・期限管理 |
| **人事** | 採用・チーム・オンボーディング |

### 4要素フレームワーク

各部署のCLAUDE.mdを以下の4要素で構成し、専門家として振る舞います:

- **ペルソナ**: 専門家としての人格定義
- **絶対原則**: 破ってはならないルール
- **能動的アクション**: 指示がなくても実行すべきこと
- **過去の教訓**: 運用で得た知見の蓄積

### リサーチパイプライン

リサーチ依頼が来たら、秘書が5ステップを確認なしで一気通貫で回します。
オーナーに見せるのはA評価が出た最終成果物のみ。

```
① 単一リサーチ → ② 粒度リサーチ → ③ レビュー → ④ 指摘反映 → ⑤ 再レビュー（A評価まで）
```

### ダッシュボード

全部署の概要をKPI付きで一覧表示。

- リサーチ本数・レビュー通過率・TODO完了率・inbox未処理数
- HTML出力オプションでブラウザで可視化

### スナップショット自動保存

- セッション終了時に全部署の状態を自動保存
- 次回セッション開始時に最新スナップショットを読み込み
- セッション間のコンテキスト維持を実現

### アラート機能

セッション開始時に自動チェック:

- デッドライン接近（TODO期限の前日に警告）
- リサーチの鮮度切れ（30日以上経過したデータにフラグ）
- 未処理inbox（5件以上溜まったら通知）
- レビュー未完了のリサーチを検出

### 週次レビュー

「週次レビュー」で今週の全部署の成果を自動集計。来週のTODO提案も自動生成。

### 実績レポート生成

指定期間の成果をまとめたレポートを自動生成。note記事やコンサル報告書として使える形式で出力。

### 意思決定フレームワーク

「判断を整理して」で意思決定支援モードに入ります:

- Pros/Cons表（メリット・デメリット整理）
- SWOT分析
- 判断基準マトリクス（複数の選択肢をスコアリング）

### 業種別テンプレート

「業種テンプレートを使う」で最適な部門構成を一括セットアップ:

- 士業（弁護士・税理士・行政書士など）
- EC・小売
- 飲食・店舗
- 不動産

### エクスポート機能

- **Notion**: Markdownのままコピペ可能な形式で出力
- **Obsidian**: `[[ファイル名]]` 形式の内部リンクを自動生成

### MCP連携

外部サービスとの連携が便利な場面で、秘書がMCPサーバーの導入を提案します。

| サービス | 用途 |
|---------|------|
| Google Calendar | スケジュール管理 |
| Notion | ナレッジ管理 |
| GitHub | Issue・PR管理 |
| Slack | チームコミュニケーション |
